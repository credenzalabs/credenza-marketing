How to export each as a transparent PNG

Best method (transparent background, exact pixel dimensions):

  1. Open this file in Chrome (or any Chromium browser).
  2. Right-click the headline you want to capture → Inspect.
  3. In DevTools, find and right-click the <div id="headline-primary"> (or headline-italic) element.
  4. Choose Capture node screenshot. A transparent PNG downloads automatically.
  5. Repeat for the second headline.

Quick method (white background, then alpha-out in Preview/Photoshop):

  1. Cmd+Shift+4 → drag a tight selection around each headline → screenshot saves to Desktop.
  2. Open in Preview → Tools → Instant Alpha → drag across the white background to remove it → File → Export as PNG.

PDF method (single document, both headlines):

  1. Cmd+P → Save as PDF. Print stylesheet hides chrome and renders both blocks on transparent backgrounds.
